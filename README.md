# kata-bankaccount
Kata Bank Account Full web

# Prerequisites
- Node (https://nodejs.org/en/)
- Install bower
```bash
> npm install -g bower
```

# Installation

##Install the project
```bash
//In project root folder
> npm install
> bower install 
```

##Build the project
```bash
//In project root folder
> gulp build
```

## Start the project
```bash
//In project root folder
> gulp connect

//It starts the project in http://localhost:8080/
```
